# IPO-Dashboard
Project Name:
IPO One-Stop Dashboard

Goal:
A single website to view IPO calendar, fundamentals, and allotment links
for informational purposes only.

Target Users:
Retail IPO investors in India

Version 1 Features:
- IPO calendar (Upcoming / Open / Listed)
- IPO fundamentals (price band, lot size, issue size)
- Company financials (revenue, profit)
- Registrar allotment redirect links
- Disclaimer page

Out of Scope (Not in v1):
- GMP
- Buy/Sell recommendations
- Predictions
- Paid features

Tech Stack:
Frontend: React + Tailwind
Backend: Node.js + Express
Database: PostgreSQL
Hosting: AWS + Vercel

Data Source:
DRHP PDFs, NSE/BSE public data

Legal Note:
Educational use only. No investment advice.
